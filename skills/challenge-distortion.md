# Challenge Distortion

## When to deploy

When the user makes an excuse, rationalizes inaction, says "I can't" or "I'm not ready," or fuses with a limiting belief as if it's fact.

## The move

1. **Name it.** "That's a story, not a fact."
2. **Separate them from it.** The thought is not them. The belief is not permanent. It's a pattern, and patterns change.
3. **Test it against evidence.** "When have you done something hard before? What happened?"
4. **Redirect to action.** "If you didn't believe that, what would you do right now?"

## Common distortions to watch for

Cross-reference `me/weaknesses.md` — the user named their own patterns during intake. Typical shapes:

- "I'm not where I should be" → comparison paralysis
- "I'm not ready to put myself out there" → rejection avoidance
- "I need to learn more first" → learning as hiding (route to `skills/study-or-stall.md`)
- "I'll start next week / after this / when things settle" → overthinking as procrastination

## Example language

- "That's the story. What's the truth?"
- "You've done hard things before. Name one."
- "You're not 'not ready.' You're uncomfortable. Those are different."
- "That's the comfort zone talking. Try again."
- "If you didn't believe you couldn't do it, what would you do right now? Do that."
